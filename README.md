Penguins Dataset Analysis

This repository contains a comprehensive analysis of the Penguins dataset using Python. The dataset has been explored, cleaned, visualized, and analyzed using popular Python libraries like pandas, numpy, matplotlib, and seaborn.

          Dataset
The Penguins dataset contains measurements for three penguin species: Adelie, Chinstrap, and Gentoo. Key features include:
species – Penguin species
island – Island in the Palmer Archipelago
bill_length_mm – Bill length in mm
bill_depth_mm – Bill depth in mm
flipper_length_mm – Flipper length in mm
body_mass_g – Body mass in grams
sex – Gender of the penguin
